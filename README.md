piston-workspace
================

Git submodules of Piston projects

## How to build Piston

```
git clone https://github.com/PistonDevelopers/piston-workspace.git
cd piston-workspace
chmod +x build.sh
./build.sh
```

## Piston '.rlib' files

The Piston '.rlib' files are located in 'piston-workspace/piston/target/cpu-vendor-os/lib'.
